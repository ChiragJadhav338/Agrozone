🌱 AgroZone – Agricultural E-Commerce Platform
📌 Project Overview

AgroZone is an agricultural e-commerce platform designed to connect farmers to single digital platform.
The goal of the project is to simplify the process of buying agricultural products online and See information of crops while providing farmers with a broader market and better accessibility.

This platform allows users to browse agricultural products, view product details, and interact with the system through a simple and user-friendly interface.
